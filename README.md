# java-sample
Sample Java code for learning.

## Building
Build with Gradle.

    # build with
    gradle build

    # run with
    gradle run

    # clean with
    gradle clean

**Note**: You may use `./gradlew` instead of `gradle` if you haven't installed
gradle already.
